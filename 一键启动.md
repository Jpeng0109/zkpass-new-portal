# 一键本地全栈（部署前自测）

## 1. 配环境（只做一次）

```powershell
# 后端（若已在 backend 目录，不要再 cd backend）
cd backend
copy .env.example .env
# 编辑 .env：填入你的 Atlas MONGODB_URI，PORT=5000

# 前端（仓库根目录）
cd ..
copy .env.local.example .env.local
```

## 2. 连云 + 灌库

```powershell
cd backend
npm install
npm run seed
```

## 3. 跑路由（API）

```powershell
npm run dev
```

另开终端验证：

```powershell
cd ..
npm run check:api
```

## 4. 前端 Axios 联调

```powershell
npm install
npm run dev
```

或一条命令：

```powershell
npm run dev:all
```

- 前端 http://localhost:8080  
- API http://localhost:5000/health
